# Microtubule-Simulator
Code for generating 3D microtubule model
INPUT
************************************************************************
xsize, ysize, zsize: size of output matrix
sim_num: number of microtubules
step_num: total step used for simulation (length?)

OUTPUT
************************************************************************
img: 3D matrix
